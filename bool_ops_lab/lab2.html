Question 1: Determine the order of operations for the boolean operators: and, or, not.

>>> x = True
>>> y = False
>>> not y and y
False

The python terminal code above proves that the boolean operator not has priority over and. This can be seen through the experiment above, if we assume and has priority then the expression y and y is evaluated first; simplifying the expression to not False. This would evaluate to True, which is contradictory to the experiment above, proving that the not operator has presidence over the and operator.

>>> x = True
>>> y = False
>>> y or y and x or x
True

Using the experiment above we proved that and has priority over or. This can be shown in the example above, if or has priority in the order of operations then the boolean expression above would evaluate to False. As we can see, the boolean expression above evaluates to True, meaning and has presidence over or.

Now we have two conditions to determine the boolean order of operations.Condition 1: not must have presidence over and. Condition 2: and must have presidence over or. Using these two conditions we can derive the boolean order of operations: not, and, or, which is the only combination of boolean operators which will satisfy the conditions above. not, and, or is the correct order of operations!
